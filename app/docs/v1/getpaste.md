# Get Paste

`GET` https://pastefy.ga/api/v1/get/%ID%

### Return

```json
{
  "exists": true,
  "id": "%ID%",
  "title": "TITLE",
  "created": "2019-11-21 13:18:04",
  "content": "CONTENTS"
}
```

### Password
`GET` https://pastefy.ga/api/v1/get/%ID%?password=%PW%
